jQuery Mosaic Gallery Plugin

How to use?

		$(function(){
			$('#gallery').imgg({
				images: ['images/img1.jpg','images/img2.jpg','images/img3.jpg','images/img4.jpg']
			});
		});

Just pass in the id of the container div and pass an array of images in the constructor.
